# AWS-Static-Web-Hosting
This repository contains my S3 static website hosting project done during my internship.

Duriation:6 Weeks
Project Name:AWS Static Website Hosting
Project Scope:This project involves hosting a static website using Amazon S3 (Simple Storage Service). It demonstrates how to deploy a simple web application consisting of index.html and error.html without using any backend server. The project includes configuring an S3 bucket, enabling static website hosting, setting public access permissions, and deploying the website for public access via a generated endpoint URL.
